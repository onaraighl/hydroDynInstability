# Spectral

This directory contains some simple codes to obtain the eigenvalues and eigenfunctions of the following equation:

$$
\frac{d^2f}{dy^2}=-\lambda f,\,\,\,\,y\in (-L/2,L/2).
$$

Homogeneous boundary conditions are assumed on either end:

$$
f(-L/2)=f(L/2)=0.
$$
